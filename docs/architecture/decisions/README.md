# Architecture Decision Records (ADRs)

Foundational, cross-cutting decisions for the **trustworthy-notes** application — the rationale future maintainers will most need. Each ADR captures **Context → Decision → Consequences**. Numbering is sequential; an ADR is *Accepted* once merged. These records cover the application's own architecture; they are distinct from the project-kit methodology decisions under `.pkit/decisions/`.
